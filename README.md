# Snare by Saedos Releases

Public binary downloads for **Snare by Saedos**, a TR-909-inspired snare drum
synthesizer plugin with modern sound-design extensions.

Download the latest packages from the
[Releases](https://github.com/ryandorward/snare-by-saedos-releases/releases)
page.

## Formats

- macOS: VST3 and Audio Unit
- Windows: VST3

## Install

On macOS, copy the VST3 and/or Audio Unit bundle into:

```text
~/Library/Audio/Plug-Ins/VST3
~/Library/Audio/Plug-Ins/Components
```

On Windows, copy the VST3 bundle into:

```text
C:\Program Files\Common Files\VST3
```

This repository contains binary releases only.
